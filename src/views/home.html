Linguagem de programação C

A linguagem de programação C criada por Dennis Ritchi. programador de hardware em 1972. Está linguagem de programação foi declara por ser de baixo nível, isso significa controle com o hardware do computador, e sendo funcional em qualquer sistema operacional. C é uma linguagem poderosa e eficiente para a aréa de desenvolvimento de software de sistema. Seu nível de apredizado pode ser difícl, mais quando nós falamos em desempenho de hardware pode aposta é C.

Quais componentes do hardware podemos manipular

  • Controle do estado de carga
  • Controle sobre a tempuratura da CPU
  • Acesso a gavetas e unidades de Memória RAM
  • Executar codigo direto no Kernel do OS
  • Informações sobre sistema

Monitor de informações Kernel

Analise do Codigo

Este é um codigo escrito em linguagem C, responsável por retornar informações do kernel, versão demo e versão completo do hardware, tipo de arquitetura e qual sistema está em uso. Instalado no hardware. Utiliza-se neste codigo modulos nativos do proprio hardware como stdio.h responsável pelam entrada e saido da resposta do programa. Por exemplo um printf. O modulo sys/utsname é para fazer a busca das informações do nosso hardware, logo em seguida, salvaremos os dados em uma variavél armazenada em unidade de memória. para mais informações sobre este codigo.

Monitor de Hardware

Analise do Codigo

  • Temperatura dos núcleos da CPU
  • Uso da memória RAM
  • Carga média do sistema operacional

Esses #include importam as bibliotecas necessárias stdio.h, stdlib.h, string.h: funçao padrão de entrada, saída, alocação de memória e manipulaçao de strings.
sensors/sensors.h: biblioteca libsensors para obter dados de sensores do sistema(como arquitetura).
unistd.h: fornece sleep() e outras funções POSIX.

Função: view_temp() Detecta os chips com sensores disponíveis no sistema sensors_get_detect_chips. Para cada chip, percorre suas "features" caracteristicas do sensor. Verifica se a features é do tipo SENSORS_FEATURES_TEMP (temperatura). Pega a subfeature de entrada de temperatura TEM_INPUT Lê o valor da temperatura com sensors_get_value() e exibe no console. para mais informações sobre este codigo.
